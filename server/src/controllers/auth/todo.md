# Endpoint Açıklama

✅ | 1 POST /auth/sign-up Yeni kullanıcı kaydı

✅ | 2 POST /auth/sign-in Giriş (email/username + şifre)

✅ | 3 POST /auth/sign-out Çıkış / token iptali

| 4 GET /auth/me Oturumdaki kullanıcı bilgisi

| 5 POST /auth/refresh-token Access token yenileme

| 6 GET /auth/verify-email Email doğrulama linki

| 7 POST /auth/resend-verification Doğrulama mailini tekrar gönder

| 8 POST /auth/forgot-password Şifre sıfırlama maili

| 9 POST /auth/reset-password Şifreyi sıfırla

| 10 DELETE /auth/delete-account Hesabı sil
